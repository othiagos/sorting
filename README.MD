# Sorting

Generates the runtime of some sorting method for a specific size.

### **Compilation**

```bash
$ make
```
### **Arguments**
```
./bin/sorting COMMAND VECTOR_SIZE SEED

```
### **Commands**
>H = heapsort \
M = mergesort \
Q = quicksort \
I = insertionsort \
S = selectionsort \
B = bubblesort

### **Run and Generate Graph**

```bash
$ python main.py
```
